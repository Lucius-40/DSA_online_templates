# DSA-2 Templates

Templates for CSE-208 Online exams.

## Directories

- **algorithms/** - Graph algorithms (Dijkstra, Bellman-Ford, Floyd-Warshall, BFS, DFS, Topological Sort)
- **AVL_Tree/** - AVL tree data structure implementation
- **hash_table/** - Hash table implementations and related problems
- **Red_Black_Tree/** - Red-Black tree data structure implementation
- **prac/** - Practice problems organized by topic:
  - APSP problems/ - All-pairs shortest path problems
  - flow/ - Network flow and matching problems
  - MST_problems/ - Minimum spanning tree problems
  - SSSP Problems/ - Single-source shortest path problems
